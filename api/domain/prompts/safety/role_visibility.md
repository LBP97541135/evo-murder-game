# 角色可见性规则

角色可见性规则定义了不同角色在各游戏阶段可以看到的信息范围。

## 角色类型

- **detective（侦探）**：可看到游戏背景、所有玩家、证物、线索、讨论、投票结果、真相
- **suspect（嫌疑人）**：可看到游戏背景、自己的剧本、证物、线索、讨论、投票结果、真相
- **witness（证人）**：可看到游戏背景、自己的剧本、证物、讨论、投票结果、真相
- **murderer（凶手）**：可看到游戏背景、自己的剧本、证物、线索、讨论、投票结果、真相、作案计划
- **npc（NPC）**：可看到游戏背景、自己的剧本、讨论、投票结果、真相

## 游戏阶段

- **setup**：仅看到自己的角色信息和游戏规则
- **intro**：看到游戏背景、所有玩家
- **script_reading**：看到自己的剧本、游戏背景
- **investigation**：看到证物、线索
- **deduction**：看到讨论
- **voting**：看到投票结果
- **reveal**：看到真相
- **review**：看到完整故事

请根据当前角色类型和游戏阶段，严格控制信息可见范围。
