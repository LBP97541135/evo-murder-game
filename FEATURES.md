## 5. 游戏主界面（Game Page）

**路径：`/play/:id`**

这是核心玩法界面，包含完整的 **6 阶段游戏循环** + 侧边栏交互系统。

### 5.1 游戏阶段

| 阶段 | ID | 说明 |
|---|---|---|
| ① 选角 | `role-selection` | 选择扮演角色，确认阵容 |
| ② 读本 | `script-reading` | 逐章阅读剧本，可高亮关键语句 |
| ③ 自我介绍 | `intro` | 按顺序完成角色自我介绍 |
| ④ 搜证 | `search` | 2 次随机搜证机会，发现新证物 |
| ⑤ 讨论 | `discussion` | 公共自由讨论，发言队列制 |
| ⑥ 投票 | `vote` | 推理投票，提交凶手判断 |

### 5.2 核心交互功能

#### 发言队列系统

- 点击「选择发言」申请进入队列
- 按申请顺序每人 90 秒发言倒计时
- 当前发言者结束后自动移交下一位
- 可「取消申请」，可「结束发言」，可「跳过 Agent」

#### 指定 Agent 回答

- 自己发言时，可将任意 Agent 指定为下一位强制回答
- 被指定 Agent 必须发言，其他角色不可插队
- 回答结束后恢复原队列

#### 私聊系统

- 点击「发起私聊」选择对象建立私聊
- Agent 可主动发起私聊申请（允许 / 稍后 / 拒绝）
- 私聊内容与证物不公开，独立于公共讨论

#### 证物系统

- 个人证物栏：查看已获得的所有证物
- 出示证物：选择公开范围（所有人 / 指定角色）+ 出示理由
- 证物详情：地点、时间、获得方式
- 已质询线索：针对证据或关键发言发起质询的问答记录

#### 搜证

- 每局 2 次随机搜证
- 搜到的新证物自动保存到个人证物栏
- 进度条可视化剩余搜证次数

#### 剧本高亮

- 在剧本文本中**拖动选中**具体语句
- 点击「高亮选中文段」保存到个人高亮列表
- 高亮内容持久化到 `localStorage`，下次打开仍可见
- 可随时删除高亮

#### 线索指向与质询

- 在公共讨论中点击任意发言，可「指向嫌疑人」+ 填写理由
- 可针对关键发言或证据发起「质询」，对方自动给出预设回答
- 质询记录保存到证物栏

#### 推理投票

- 满足后端条件后方可进入投票阶段
- 选择嫌疑人 + 填写推理理由 + 勾选关键证物
- 提交后不可更改，等待其他玩家完成推理

### 5.3 右侧边栏（Tabs）

| Tab | 内容 |
|---|---|
| 剧本 | 打开个人剧本弹窗（含高亮管理） |

游戏页通过 `api/invoke.ts` 连接后端，支持：

- 创建 / 恢复游戏会话
- 获取 / 强制切换游戏阶段
- 提交投票
- 保存 / 拉取对话历史
- 获取证物、保存证物、出示证物
- 结合证物进度追踪
